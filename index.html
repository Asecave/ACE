Was ist die ACE?

Mit der Asecave Engine können Sie ganz einfach und schnell projecte erstellen. Sie bietet die folgenden bibliotheken:

ACW

Mit Asecave Windows können Sie mit nur einer Zeile Code ein ganzes Fenster erstellen! Asecave Windows erleichtert es ihnen ein Fenster zu erstellen und zu konfigurieren.

ACI

Asecave Input können Sie ohne Probleme Eingabegeräte wie Tastatur und Maus abfragen. Mit der Asecave Windows Bibliothek kombiniert ist diese Bibliothek sehr mächtig.

ACL

Asecave List ist die verbesserte Version aller Java List-Klassen. Mit integriertem Quicksort-Algorythmus ist die Asecave List Bibliothek schnell und optimiert.

ACA

Asecave Audio liefert einen Audioplayer der einfach zu nutzen ist. Sound importieren, die Sound-Klasse lesen lassen, play ausführen und schon spielt die Musik!

All das liefert die ACE!
